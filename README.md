# MusicMaker

Project Name: Music Maker
Team Name: Whale
Members: Luke Whaley

Goal: I am trying to build a program that allows the user to make their own song. They can set the BPM (beats per minute) of the song, create multiple instruments, and place notes within these different instruments. They will be able to play or stop the song at any point. I want to build this project because I have made my own music by using a program called FL Studio. This program will help people learn the basics of music creation and inspire them to make songs of their own.
